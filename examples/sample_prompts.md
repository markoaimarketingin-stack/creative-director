# Sample Prompts

## Hook generation

```text
You are Creative Director Engine, a senior direct-response creative strategist.
Generate 15 high-converting ad hooks for Marko AI.
Audience: startup founders and growth marketers.
Focus on curiosity gaps, emotional triggers, and direct-response marketing principles.
Cover curiosity, fear-based, benefit-driven, contrarian, and social-proof hooks.
Return structured output only.
```

## Messaging angle generation

```text
Generate 5 distinct messaging angles for Marko AI.
Each angle must include a name, description, target emotion, and use case.
Make the angles suitable for Meta conversion campaigns targeting startup founders.
Return structured output only.
```

## Ad copy generation

```text
Generate 20 Meta ad copy variants using the supplied hooks and messaging angles.
Keep primary text under 125 characters where possible, headlines under 40, and descriptions under 30.
Every record must include hook_text, angle_name, primary_text, headline, CTA, and description.
Return structured output only.
```

## Visual concept generation

```text
Generate 5 visual concepts that can be sent to an image generation model.
Each concept must include hook_text, angle_name, scene_description, camera_angle, background_setting,
color_palette, mood, style_reference, aspect_ratio, and media_type.
Prefer scroll-stopping commercial scenes over abstract brand imagery.
Return structured output only.
```
